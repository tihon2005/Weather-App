## Weather App

- Written in Swift
- Built in Xcode 14
- Uses SnapKit
- Uses UserDefaults
- Uses API
- iOS 14.0+

## About

An app that shows temperature, wind speed and wind direction for London, Berlin and Moscow using API and UserDefaults.
